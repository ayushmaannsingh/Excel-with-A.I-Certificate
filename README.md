





  
  # Excel-with-A.I-Certificate 
Key AI Features in Excel
Microsoft 365 Copilot: Acts as an AI assistant to analyze data, create charts, and format tables via a chat pane.
Analyze Data: A built-in feature that answers questions about datasets, creates visualizations, and finds trends using natural language queries.
Formula Generation: Tools like GPT Excel and AI formula add-ins can create complex formulas, SQL queries, and scripts from simple instructions.
Data Cleaning: AI can automatically identify duplicates, fix inconsistent capitalization, and format data.
Data Extraction from Pictures: Converts images of printed tables into editable Excel spreadsheets.
AI Agent Mode (Excel Labs): A more advanced feature that handles complex, multi-step tasks, such as creating full, interactive dashboards and managing entire workbooks.
